# Challenging_Project
참여해볼 해외 프로젝트 명단들 

1) George B. Moody PhysioNet Challenge 2026 — 헬스케어
- 수면다원검사(PSG)와 기본 인구통계 정보를 바탕으로, 환자의 향후 인지장애 진단 여부/시점 예측하는 챌린지.
- 2026년 대회는 이미 오픈됨. 비공식 단계 : 2월 10일~4월 9일, 공식 단계:  5월 11일~8월 말 일정

2) SNOMED CT Entity Linking Benchmark — 헬스케어/NLP
- 임상 노트 안의 텍스트 span을 SNOMED CT 의료 개념에 연결하는 벤치마크.
- DrivenData 메인 페이지 기준으로 Benchmark 형태로 운영 중이고, 10개월 이상 남아 있는 상태로 표시되어 있음.
- 임상 NLP, entity linking, medical LLM 파이프라인 연습용으로 아주 좋음.

3) Flu Shot Learning: Predict H1N1 and Seasonal Flu Vaccines — 헬스케어/입문형
- 사람들의 배경, 인식, 건강행동 데이터를 바탕으로 독감 백신 접종 여부 예측
- DrivenData에서 beginner practice로 운영 중이고 1년 이상 남아 있음으로 표시돼 있음.

4) DengAI: Predicting Disease Spread — 헬스케어/시계열
- 기상·환경 데이터를 이용해 주별 뎅기열 환자 수를 예측하는 대회.
- 현재 DrivenData에서 학습용으로 열려 있고, 종료일: 2027년 10월 5일 11:59 p.m
(헬스케어지만 사실상 시계열 예측 + 외생변수 활용 훈련용으로 아주 좋음.)

5) Youth Mental Health Narratives: Automated Abstraction — 헬스케어/LLM
- 청소년 자살 관련 narrative에서 표준 변수를 자동 추출하는 대회였음, 2024년 11월 종료되었으나 멘탈헬스 + 의료 텍스트 + 정보추출 포트폴리오용으로 좋음. 

6) U.S./U.K. PETs Prize Challenge – Financial Crime Track — 금융/프라이버시 보존 ML
- 연구형 플젝, 금융 범죄 탐지를 위해 privacy-preserving federated learning 솔루션을 개발하는 챌린지
  
7) Zimnat Insurance Recommendation Challenge — 금융/추천시스템
- 보험사 고객 데이터 바탕으로 어떤 보험 상품 추천할지 예측하는 대회.
- 약 4만 명 규모의 고객 데이터가 제공되었고, 금융 서비스에서 추천시스템 문제를 다룬다는 점이 재밌음. 금융 도메인인데도 마케팅 추천과 닿아 있어서 포폴 활용도가 높음.

8)Absa Customer Income Prediction Challenge — 금융/고객 분석
- 거래 이력으로 고객 소득을 예측하는 챌린지
- Zindi 페이지 기준으로 2022년 11월 29일 시작, 2023년 2월 26일 종료, 상금 $5,000, 다만 남아공 참가자 한정 조건이 있었음. 지금 새로 참가하는 용도보다는, 금융 고객 데이터 기반 feature engineering 연습용으로 좋음.
  
9) Adbot Ad Engagement Forecasting Challenge — 마케팅/광고 예측
- 디지털 마케팅에서 광고가 앞으로 1주·2주 후 몇 번 클릭될지 예측하는 대회.
- 단순 예측만이 아니라 클릭에 가장 영향을 준 변수 5개를 같이 제출해야 해서, 설명가능성까지 신경 쓰게 만드는 점이 어려움.
- 광고성과 예측, 마케팅 믹스 감각, 시계열/회귀 쪽 포트폴리오로 좋음.

11) Farm to Feed Shopping Basket Recommendation Challenge — 마케팅/리테일 추천
- 과거 거래 데이터를 바탕으로 미래 쇼핑 바스켓을 예측해서 추천하는 문제.
- Zindi 페이지 기준으로 2025년 12월 2일 시작, 2026년 1월 19일 종료, 상금은 €8,250, “recommend future shopping baskets”가 핵심 과제.
